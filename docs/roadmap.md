# Roadmap

These items are **planned or in progress** and are **not yet live features**. They are listed
separately from the implemented system so there is no ambiguity between current state and intent.

## In progress

- **Dedicated home-security system** — the central next build-out; an expanded security capability
  beyond the current automation stack.

## Planned

- **Advanced security features** — additional hardening and security tooling layered onto the platform.
- **AI integration** — incorporating AI capabilities into the system.
- **Monitoring of multiple automations and bots** — extending observability to cover a broader set of automations and bots running on the platform.
- **Backups and restore validation** — implementing backups of configuration and data, and testing a full restore.
- **Container-level hardening** — running containers as non-root and/or with read-only filesystems.

## Known constraints

- The **legacy host hardware (Dell XPS 410)** makes installing newer software difficult and is the main limiter on both performance and reliability. Hardware modernization is the implicit dependency under several roadmap items.
